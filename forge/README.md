# Forge Branding Assets

Forge is the **Interface Compiler** of the System Zero ecosystem. It represents Structure (Iron) + Transformation (Heat).

## Design Rationale
- **The Metaphor**: Structure + Heat.
- **The Shape**: Minimalist "F" constructed from three blocks.
- **The Colors**:
  - `Magma Orange` (#FF3D00 - #FF9100): Represents Creation, Design, Energy, AI.
  - `Void Bg` (#090A0B): Represents the System background.
  - `Surface` (#161B22): Represents UI panels.

## Typography
- **Headings/Body**: System Stack (`-apple-system`, `Inter`, `Roboto`).
- **Code**: `JetBrains Mono` or `Fira Code`.

## Vocabulary
- **Don't say**: "Draw", "Sketch", "Canvas", "Artboard".
- **Do say**: "Define", "Compile", "View", "Frame", "Layout Engine".
